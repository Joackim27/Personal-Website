Arduino based Security Radar System

This is a simulation of the project titled “A Security Radar System on a Semi-Autonomous Car based on the Ultrasonic Sensor Servo Motor and Arduino Uno.” The research was developed using an ultrasonic sensor (HC-SR04), servo motor, piezo buzzer, red and green LED, and an Arduino Uno R3 for the microcontroller. This radar system detects objects by utilizing the ultrasonic sensor that is attached to the server motor. The servo motor moves the ultrasonic sensor from 0 to -90 degrees, -90 to 0 degrees, and 0 to 90 degrees simulating a -180 to 180 rotation of the servo motor for the ultrasonic sensor. If the ultrasonic sensor detects an object below 30 cm distance, Arduino Uno R3 activates the buzzer and the Red LED. On the other hand, if the object is 30 cm above, the Arduino Uno R3 activates the green LED and no buzzer sound. Finally, the distance of an object by the ultrasonic sensor can be seen in the serial monitor.
